Terminal

Raw shell access (always available)

Built for how you actually work

No electron apps, no desktop clients. Just a fast web interface that runs anywhere.

⚡

Real-time streaming

Full xterm.js terminal with 256-color ANSI support. Low-latency WebSocket streaming — it feels native.

⚙

Multi-session

Run Claude in one tab, Copilot in another. Switch between sessions. Sessions persist across restarts.

🔒

Secure by default

Token-based auth generated on startup. HTTPS support. No data leaves your machine unless you want it to.

🌐

Remote access

Built-in Microsoft Dev Tunnels integration. Share your AI terminal securely with --tunnel.

📱

Mobile-friendly

Responsive design with touch-optimized controls. Installable as a PWA on any device.

🚀

Zero config

Auto-detects installed tools. Picks up your shell, working directory, and environment. Just run it.

Up and running in seconds

Run instantly

npx ai-or-die

Opens your browser automatically with a secure auth token.

Or install globally

npm install -g ai-or-die

Common options

# Custom port
ai-or-die --port 8080

# Your own auth token
ai-or-die --auth my-secret-token

# Secure remote access
ai-or-die --tunnel

# Development mode
ai-or-die --dev
